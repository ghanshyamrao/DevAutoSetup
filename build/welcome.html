DevOnboard

This wizard will install DevOnboard on your computer.

This app helps you install development software (browsers, IDEs, runtimes, tools) in one go using Windows Package Manager (winget).

  • Select software from the catalog and install in batch
  • Uses winget for silent, reliable installs
  • Optional Supabase-backed catalog for teams

Requirements: Windows 10/11, winget, administrator rights, and an internet connection.

Click Next to choose the install location and continue.
